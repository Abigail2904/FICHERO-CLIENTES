# Clientela App

Aplicación simple para gestionar clientes con búsqueda por nombre/apellido y CRUD. Usa MongoDB.

Pasos rápidos:

1. Instalar dependencias:

```bash
npm install
```

2. Crear `.env` con la variable `MONGO_URI` (puedes partir de `.env.example`).

3. Ejecutar en desarrollo:

```bash
npm run dev
```

4. Abrir `http://localhost:3000` en el navegador.
